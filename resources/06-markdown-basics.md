# Markdown Basics

## What Is Markdown?

**Markdown** is a simple way to format text using plain characters. Instead of clicking bold buttons or menu options, you type symbols like `#`, `*`, and `-` to create headings, bold text, and lists.

Files written in Markdown use the `.md` extension (like `README.md` or `SKILL.md`).

**Why it matters for the portfolio project:**
- Every GitHub repo uses a `README.md` as its front page
- Claude skills are written in Markdown (SKILL.md)
- Documentation, notes, and guides are often Markdown files
- This very document is written in Markdown

## Quick Reference

### Headings

```markdown
# Heading 1 (biggest)
## Heading 2
### Heading 3
#### Heading 4
```

### Text Formatting

```markdown
**bold text**
*italic text*
~~strikethrough~~
`inline code`
```

### Lists

```markdown
- Item one
- Item two
- Item three

1. First item
2. Second item
3. Third item
```

### Links

```markdown
[Link text](https://www.example.com)
```

### Images

```markdown
![Alt text](path/to/image.png)
```

### Code Blocks

Use triple backticks to create a code block:

````markdown
```html
<h1>Hello World</h1>
```
````

### Blockquotes

```markdown
> This is a quote or callout
```

### Tables

```markdown
| Column 1 | Column 2 | Column 3 |
|----------|----------|----------|
| Row 1    | Data     | Data     |
| Row 2    | Data     | Data     |
```

### Horizontal Rule

```markdown
---
```

## Preview Markdown in VS Code

1. Open any `.md` file in VS Code
2. Press `Cmd + Shift + V` (Mac) or `Ctrl + Shift + V` (Windows/Linux)
3. A preview panel opens showing how the formatted text will look
4. For side-by-side editing: `Cmd + K, V` (Mac) or `Ctrl + K, V` (Windows/Linux)
