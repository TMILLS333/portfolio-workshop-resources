# What Is VS Code?

## The Short Version

**Visual Studio Code (VS Code)** is a free code editor made by Microsoft. Think of it as a souped-up text editor — like Google Docs, but built for writing code instead of essays.

It's not the same as "Visual Studio" (that's a different, much heavier program). VS Code is lightweight, fast, and works on Mac, Windows, and Linux.

---

## Why Designers Use It

- **It's free.** No subscriptions, no hidden costs.
- **It works with everything.** HTML, CSS, JavaScript, Markdown, JSON — whatever you need.
- **Built-in terminal.** You can run terminal commands without leaving the app.
- **Built-in Git.** You can commit and push code without touching the command line.
- **Extensions.** Like plugins in Figma — you can add tools for live preview, color pickers, formatting, and more.
- **AI-friendly.** Claude Code and many AI tools are designed to work inside VS Code.

## VS Code vs. Other Tools You Might Know

| Tool | What It Is | How VS Code Compares |
|------|-----------|---------------------|
| **Figma** | Design tool | VS Code is for code; Figma is for visuals. They complement each other. |
| **Notepad / TextEdit** | Basic text editors | VS Code does everything they do, plus syntax highlighting, auto-complete, and extensions |
| **Sublime Text / Atom** | Other code editors | Similar category, but VS Code has the largest ecosystem of extensions and AI integrations |
| **Cursor** | AI-powered code editor | Built on top of VS Code — very similar interface, but with deeper AI integration |

## How to Install

1. Go to [code.visualstudio.com/download](https://code.visualstudio.com/download)
2. Download the version for your operating system (Mac, Windows, or Linux)
3. Install it like any other app
4. Look for the **blue icon** — not the purple one (that's Visual Studio, a different program)
