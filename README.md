# Portfolio Workshop — Pre-Work Resources

A curated collection of beginner-friendly resources to help designers get comfortable with the tools we'll use in our portfolio project workshop.

**No prior coding experience needed.** Each topic includes a recommended video or article, plus a quick-reference summary so you can revisit key concepts anytime.

---

## Topics

| # | Topic | What You'll Learn |
|---|-------|-------------------|
| 1 | [Terminal Basics](resources/01-terminal-basics.md) | What the terminal is, common commands, and how to navigate your computer without clicking |
| 2 | [VS Code Basics](resources/03-vs-code-basics.md) | Opening projects, creating files, using the integrated terminal, and essential shortcuts |
| 3 | [Git Basics](resources/04-git-basics.md) | What Git is, and what terms like commit, push, pull, merge, and branch actually mean |
| 4 | [Cloudflare Basics](resources/05-cloudflare-basics.md) | What Cloudflare is, why we use Cloudflare Pages, and how deploying a site works |

---

## How to Use This Repo

- **Before the workshop:** Watch or read the recommended resource for each topic. You don't need to memorize anything — just get familiar with the concepts.
- **During the workshop:** Use the quick-reference sections as a cheat sheet when you need a reminder.
- **After the workshop:** Come back anytime to revisit topics or explore the additional resources.
